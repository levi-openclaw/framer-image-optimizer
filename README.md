## How to Install

1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" ‚Üí Upload `plugin.zip`

## How to Use

### 1. Add Images
Drag and drop images or click to browse. Supports PNG, JPEG, WebP, GIF.

### 2. Configure Settings
- **Quality** ‚Äî 10-100% (lower = smaller file)
- **Max Width** ‚Äî Resize to 800px, 1200px, 1920px, or 2560px
- **Format** ‚Äî WebP (recommended), JPEG, PNG, or Original

### 3. Optimize
Click **Optimize** on individual images or **Optimize All** for batch.

### 4. Download
Download optimized images individually or view total savings.
