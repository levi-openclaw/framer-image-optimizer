Compress and optimize images for your Framer projects. Reduce file sizes without losing quality.

## What It Does

**Framer Image Optimizer** helps you:
- Compress images to reduce file size
- Convert between formats (WebP, JPEG, PNG)
- Resize images to optimal dimensions
- See before/after file sizes
- Batch optimize multiple images

## How to Use

### 1. Install
1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" ‚Üí Upload `plugin.zip`

### 2. Add Images
1. Open the plugin in Framer
2. Click **Start Optimizing**
3. Drag & drop images OR click to browse
4. Supports PNG, JPEG, WebP, GIF

### 3. Configure Settings
- **Quality** - 10-100% (lower = smaller file)
- **Max Width** - Resize to 800px, 1200px, 1920px, or 2560px
- **Format** - Convert to WebP, JPEG, PNG, or keep original

### 4. Optimize
1. Click **Optimize All** or optimize one-by-one
2. Watch the savings accumulate
3. Download optimized images

## Why Optimize?

| Format | Typical Savings |
|--------|-----------------|
| JPEG ‚Üí WebP | 25-35% smaller |
| PNG ‚Üí WebP | 60-80% smaller |
| Quality 80% | 40-60% smaller |

Smaller images = faster loading = better UX + SEO
